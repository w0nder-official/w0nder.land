# Initialization

### install packages

#### Install brew

```shell
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

#### Install library

```shell
make initialize
```

### Run

```shell
yarn dev
```

### Lint

```
yarn lint
```
